# Monorepo

## 1. Monorepo Tools

- [**Lerna**](https://lerna.js.org/)**:** Manages multi-package repositories, simplifying versioning and releases.
- [**Yarn Workspaces**](https://yarnpkg.com/)**:** Facilitates dependency management across JS/TS packages, optimizing linking and installation.
- [**Nx**](https://nx.dev/)**:** Supports JS/TS, providing tools for building, testing, and sharing libraries.
- [**Bazel**](https://bazel.build/)**:** General-purpose tool supporting multiple languages, designed for large-scale codebases, optimizing build and test performance.
- [**Rush**](https://rushjs.io/)**:** Manages JS/TS multi-package repositories, aiding versioning, release management, and changelog generation.
