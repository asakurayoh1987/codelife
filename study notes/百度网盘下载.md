# 百度网盘下载

## 安装chrome油猴扩展

https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=zh-CN

## 安装油猴脚本

源站：

* http://userscripts-mirror.org/
* https://openuserjs.org/ 
* https://greasyfork.org/

网盘助手：

* https://greasyfork.org/zh-CN/scripts/378301

另一个网盘助手

* https://www.baiduyun.wiki/

## 安装chrome aria2扩展

https://chrome.google.com/webstore/detail/aria2-for-chrome/mpkodccbngfoacfalldjimigbofkhgjn?utm_source=chrome-ntp-icon

## 安装并启动aria2c

https://github.com/aria2/aria2

```bash
aria2c --enable-rpc --rpc-listen-all
```

## 开始下载

* 资源保存至自己的百度网盘
* 访问web端的百度网盘，找到资源，此时会显示网盘助手，点击生成链接后右键选择使用Aria2下载
