# Github搜索语法

## 官方说明文档

[Searching on GitHub](https://docs.github.com/en/github/searching-for-information-on-github/searching-on-github)
[Searching for repositories](https://docs.github.com/en/github/searching-for-information-on-github/searching-for-repositories)

## 常用

* 根据star数量，例：stars:>=10000
* 根据语言，例：language:javascript
* 在README中搜索关键字，例：中文书籍 in:readme
* 根据项目组织，例：org:google
* 根据用户，例：user:torvalds language:C
* Awesome系列，例：Awesome java
