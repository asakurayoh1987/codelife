# Vim 使用

## 批量替换

```bash
# :%s#需要替换的内容#替换的内容
# 用'#'或'/'都可
# 比如如下，将'https://itunes.apple.com/cn/app/id1342633958?mt=8'替换为'https://apps.apple.com/cn/app/id1469010110'
:%s #https://itunes.apple.com/cn/app/id1342633958?mt=8'#https://apps.apple.com/cn/app/id1469010110

s 查找当前行
%s 全文查找，要全局替换，还要在末尾添加上'/g'
```

[更多关于查找和替换的内容](https://www.cnblogs.com/huxinga/p/7942194.html)

## 光标位置

- 跳转至第一行
  - :0 或:1
  - gg
- 跳转至最后一行

  - :$
  - shift + g

- 跳至行首

  - 需要按行快速移动光标时，可以使用键盘上的编辑键 Home，快速将光标移动至当前行的行首。除此之外，也可以在命令模式中使用快捷键"^"（即 Shift+6）或 0（数字 0）

- 跳至行尾

  - 快速移动光标至当前行的行尾，可以使用编辑键 End。也可以在命令模式中使用快捷键"$"（Shift+4）。与快捷键"^"和0不同，快捷键"$"前可以加上数字表示移动的行数。例如使用"1$"表示当前行的行尾，"2$"表示当前行的下一行的行尾

- 跳转到指定列
  普通模式下输入数字+“|”跳转到数字指定的列
- 跳转到下一个字符串的位置
  普通模式下输入“w”

## 复制指定的多行

比如：复制第22到43行，粘贴到45行

- 方法一：控制模式下输入22, 43 copy 45
- 方法二：
  - 光标移到结束行，输入ma
  - 光标移到开始行，输入y'a
  - 光标移到复制的行，输入p，行前复制则输入大写的P
  
- 方法三：
  - 光标移到开始行
  - 假设从当选行复制到136行，则输入：y136G

## 删除指定范围的行

- 比如删除3~5行，在命令模式下：3,5d
