# 正则片段

## 捕获分组、非捕获分组

使用"()"可以对匹配进行单独的捕获并进行操作，通过"(?:)"表示对当前分组不进行捕获操作

## 零宽断言

### 1. (?=exp)：零宽度正预测先行断言，它断言自身出现的位置的后面能匹配表达式exp。

/\d(?=px)/g

1pt <span style="color:red;">2</span>px 3em <span style="color:red;">4</span>px

### 2. (?<=exp)：零宽度正回顾后发断言，它断言自身出现的位置的前面能匹配表达式exp（javascript不支持）

/(?<=px)\d/g

px<span style="color:red;">1</span> pt1 em1 px<span style="color:red;">4</span>

### 3. (?!exp):零宽度负预测先行断言，断言此位置的后面不能匹配表达式exp。

/\d(?!px)/g

<span style="color:red;">1</span>pt 2px <span style="color:red;">3</span>em 4px

### 4. (?<!exp)：零宽度负回顾后发断言来断言此位置的前面不能匹配表达式exp（javascript不支持）

/(?<!px)\d/g

px1 pt<span style="color:red;">1</span> em<span style="color:red;">1</span> px4
